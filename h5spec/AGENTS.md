# H5 Spec

Instructions for AI coding assistants using H5Spec for spec-driven development.

## Workflow

### Stage 1: Spec

制定项目的规范约束。

### Stage 2: Explore

探索现有的代码库。

### Stage 3: Analyze

分析业务需求、UI 和交互。

### Stage 4: Design

设计技术方案，设计模式。

### Stage 5: Task

拆分任务，将设计方案拆分为可执行的任务。

### Stage 6: Coding

编写代码，根据任务，实现页面模块，组件，功能点，并遵循代码规范。
